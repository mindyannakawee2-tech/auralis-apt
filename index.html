Absolute Auralis APT Repository

This page hosts update packages for Absolute Auralis OS.

Add repository

sudo tee /etc/apt/sources.list.d/auralis.list <<'REPO'
deb [trusted=yes] https://YOUR_USERNAME.github.io/auralis-apt noble main
REPO

sudo apt update
